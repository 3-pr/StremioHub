## ✨ الميزات

<div dir="rtl">

- **🌐 تكامل عبر المواقع:** إضافة أزرار ذكية "حفظ في المكتبة" و"فتح في Stremio" مباشرة داخل مواقع استكشاف الأفلام.
- **🖱️ البحث السريع بالزر الأيمن:** حدد اسم أي فيلم أو مسلسل، اضغط كليك يمين، وابحث عنه فوراً في Stremio.
- **📚 إدارة شاملة للمكتبة:** تصفح، فلترة، ترتيب، وتحديد الأعمال كمقروءة (أفلام، مسلسلات، وقيد المشاهدة) بسهولة من نافذة الإضافة.
- **🎨 واجهة زجاجية فاخرة:** تصميم داكن مستوحى من نظام Apple بتأثير (Glassmorphism) مع حركات وتفاعلات سلسة جداً.
- **🔗 نظام الإضافات الذكي (Smart Add-ons):** قراءة إضافاتك المثبتة (مثل TMDB) واختيار المفضلة لديك لجلب قصة وتفاصيل العمل بشكل حصري.
- **🌍 دعم ثنائي اللغة:** دعم كامل للغتين العربية والإنجليزية مع محاذاة أصلية لليمين (RTL) وتبديل فوري.
- **🛠 تخصيص مرن:** اختر شكل نافذة التفاصيل (بطاقة عائمة أو شاشة كاملة)، تحكم بحجم الإضافة، والمزيد!

## 🚀 متجر إضافات كروم (قريباً)

<div dir="rtl">
نقوم حالياً بتجهيز StremioHub للإطلاق الرسمي في سوق Chrome الإلكتروني.
قريباً ستتمكن من تثبيتها بضغطة زر واحدة والحصول على التحديثات التلقائية! ابقوا معنا. ⏳
</div>

## 🛠 طريقة التثبيت (وضع المطور)

<div dir="rtl">
حتى يتم الإطلاق الرسمي، يمكنك تثبيت الإضافة يدوياً كالتالي:

1. **حمّل الإصدار الأخير:** اذهب إلى [صفحة الإصدارات (Releases)](https://github.com/3-pr/StremioHub/releases)، وقم بتحميل ملف `.zip` الخاص بآخر إصدار، ثم قم بفك ضغطه في مجلد.
   *(أو يمكنك استنساخ المستودع باستخدام Git):*
   ```bash
   git clone https://github.com/3-pr/StremioHub.git
   ```
2. افتح متصفح جوجل كروم واذهب إلى الرابط `chrome://extensions/`.
3. فعّل **وضع المطور (Developer mode)** من الزاوية العلوية.
4. اضغط على زر **Load unpacked** واختر مجلد `StremioHub` الذي قمت بفك ضغطه.
5. ثبّت الإضافة في شريط الأدوات، سجل دخولك بحساب Stremio، واستمتع!
</div>

## 🔐 الأمان والخصوصية

<div dir="rtl">

- **لا نحفظ كلمات المرور:** يتم استخدام معلومات دخولك لمرة واحدة فقط لجلب مفتاح الأمان `authKey`.
- **تخزين محلي فقط:** تُحفظ جميع البيانات وإعداداتك محلياً بشكل آمن في متصفحك باستخدام `chrome.storage.local`.
- **لا تتبع:** لا نقوم بتتبع مكتبتك، أو بحثك، أو نشاطك على الويب إطلاقاً.

</div>
